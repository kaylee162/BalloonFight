## Game Controls
D-pad for movement left, right and floating movement. No down button, character is always floating downward, unless they are on a platform. 
Button B - shoot bullets. 
Press Start to play.

# Notes to add
Created the tileset, tilemaps, spritesheet, and state screens by hand. All draw by me, its my original work. 
For the floating movement, I choose to use the up arrow on the D-pad instead of button A (what's expected in the PDF). It's easy to change it to button A to float, but I felt it was better gameplay to use the up arrow.
